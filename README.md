# Box Collector (s&box)

Проект многопользовательской игры в жанре Top-Down.

## Как начать работу

1. **Склонируйте репозиторий:**
   Откройте терминал или PowerShell и выполните:
   ```bash
   git clone https://github.com/FokusNIK9/snbox.git
   ```

2. **Откройте проект:**
   Добавьте папку `snbox` в редактор s&box.

## Как обновлять проект

Каждый раз, когда в репозиторий пушатся обновления, просто запустите файл:
- **update.bat** (двойным кликом)

Он автоматически скачает все изменения через `git pull`.
