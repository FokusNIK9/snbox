# Dialogue from Zero

Эта папка — наша рабочая папка для архитектурных заметок и решений, которые мы обсуждаем с нуля.

## Правила рабочей папки

- Все важные решения по архитектуре, multiplayer, s&box и структуре проекта фиксируем здесь.
- Не записываем сюда каждый короткий чат дословно; сохраняем только полезные выводы, договорённости и планы.
- Все заметки должны быть практичными: чтобы потом по ним можно было реализовывать код в проекте.
- Официальная информация по s&box сначала проверяется по документации/API, потом используется как основа.

## Рабочий контекст

- Проект: Box Collector
- Репозиторий: FokusNIK9/snbox
- Рабочая папка заметок: docs/chat_from_zero/
- Основной файл журнала: docs/chat_from_zero/dialogue_log.md

## Зафиксированное решение

Мы договорились, что `docs/chat_from_zero/` — это наша рабочая папка для всего, что мы с тобой обсуждаем “с нуля”: архитектура, сетевой multiplayer, структура игровых систем, решения по компонентам и планы реализации.
